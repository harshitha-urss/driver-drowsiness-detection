Universal Driver Drowsiness Detection System

Real-Time Monitoring Using OpenCV, Flask & Haar Cascade

View Source Code

Problem Statement

Road accidents due to driver fatigue are a major global concern. Continuous monitoring of driver alertness is essential to prevent loss of life. Manual monitoring is not reliable, hence an automated real-time drowsiness detection system is required.

Project Demo Video

System Architecture

Working Process

  • Webcam captures real-time video
  • Face detected using Haar Cascade
  • Eye detection from face region
  • Fatigue score calculated
  • Alarm triggered if eyes closed beyond threshold

Technologies Used

Python | OpenCV | Flask | Haar Cascade | Pygame | HTML | CSS | GitHub

Results

The system successfully detects eye closure in real time and triggers an alert sound. It works accurately under normal lighting conditions with minimal CPU usage.
